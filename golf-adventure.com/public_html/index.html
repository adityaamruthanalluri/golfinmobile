Om City Network

City Network levererar prisvärda domän- och webbhotells-tjänster som passar både för företag och privatpersoner. Med ett egenägt datacenter i Sverige med full redundans säkrar vi driften av er dedikerade server eller virtuella server och webbplats. Ni registrerar er domän hos oss till ett av Sveriges absolut bästa pris och kan till detta enkelt lägga till ett prisvärt webbhotellspaket.

Du kan enkelt växa med oss. Samla dina domäner på bästa sätt. Kör din blogg eller hemsida via vårt webbhotellspaket. Bygg vidare om du behöver mer kraft via virtuella servrar (Cloud Computing) eller dedikerade servrar. Allt för din satsning online.

Vår 60 dagars pengarna-tillbaka-garanti låter er prova oss utan risk. Fungerar det inte som ni tänkt er får ni alltså pengarna tillbaka.
